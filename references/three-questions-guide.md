# 三问法深度指南

## 核心理念

三问法源自价值投资的基本逻辑：买股票就是买公司，买公司必须买自己懂的公司。

这三个问题覆盖了投资决策的三个核心维度，缺一不可：

| 问题 | 覆盖维度 | 缺失风险 |
|------|---------|---------|
| 为什么好？ | 公司质地 | 买了垃圾公司 |
| 为什么便宜？ | 估值合理性 | 买贵了 |
| 为什么现在买？ | 时机催化 | 永远等不到 |

---

## 第一问：你为什么觉得这个公司好？

### 目标

逼出具体的**盈利逻辑**和**护城河来源**，而非笼统的"行业好""品牌强"。

### 高质量回答的特征

- 能说清公司**怎么赚钱**（收入来源、成本结构）
- 能说明**为什么能持续赚钱**（护城河：品牌、网络效应、成本优势、牌照壁垒）
- 能解释**客户为什么不离开**（转换成本、粘性来源）

### 追问策略

| 用户回答 | 追问方向 |
|---------|---------|
| "它是行业龙头" | "龙头地位体现在哪？市占率多少？为什么别人抢不走？" |
| "品牌护城河很深" | "品牌护城河体现在财务数据上是什么？毛利率比同行高多少？" |
| "它的技术很领先" | "技术领先能持续多久？研发投入占比多少？专利壁垒在哪？" |
| "这个行业前景很好" | "行业好不等于公司好。这家公司独特的竞争优势是什么？" |

### 认知评估标准

| 分数 | 标准 |
|------|------|
| 8-10分 | 能说清盈利模式、护城河来源、客户粘性，有具体数据支撑 |
| 5-7分 | 能说出竞争优势，但缺少数据或逻辑不够完整 |
| 1-4分 | 只说"行业好""品牌强""大家都在用"，没有实质内容 |

---

## 第二问：你为什么觉得这个公司便宜？

### 目标

逼出具体的**估值方法**和**参照系**，而非只看PE高低。

### 高质量回答的特征

- 明确使用的**估值方法**（PE/PB/PS/DCF/可比公司）
- 有**历史估值区间**的参照（当前处于什么百分位）
- 有**同行对比**（为什么它比同行便宜/贵得合理）
- 理解**估值陷阱**（低PE可能是因为增长停滞，高PE可能是因为增长快）

### 追问策略

| 用户回答 | 追问方向 |
|---------|---------|
| "PE才15倍，很便宜" | "15倍相对于它的历史估值是什么水平？相对于同行呢？" |
| "比同行便宜" | "同行PE多少？它便宜是因为被低估，还是本来就该便宜？" |
| "增速30%，这个PE很值" | "那就是PEG=0.5。你算过它的增速能持续多久吗？" |
| "PB低于1" | "PB低可能是因为ROE低。它的ROE是多少？为什么低？" |

### 认知评估标准

| 分数 | 标准 |
|------|------|
| 8-10分 | 能用具体指标谈估值，知道历史估值区间，理解估值陷阱 |
| 5-7分 | 有估值概念，但缺少历史或同行参照 |
| 1-4分 | 只看PE高低，不懂估值逻辑 |

---

## 第三问：你为什么觉得这个公司现在买？

### 目标

逼出具体的**催化剂**和**时机逻辑**，而非"感觉要涨"。

### 高质量回答的特征

- 有**明确的催化剂**（业绩拐点、新产品、政策变化、管理层变动）
- 有**时间框架**（催化剂什么时候兑现？）
- 有**触发条件**（什么信号出现说明判断正确/错误？）

### 追问策略

| 用户回答 | 追问方向 |
|---------|---------|
| "感觉要涨了" | "你的感觉基于什么？有什么具体的事件或数据支撑？" |
| "有政策利好" | "什么政策？什么时候出台？对业绩的影响能量化吗？" |
| "机构在买" | "哪些机构？买了多少？这是领先指标还是滞后指标？" |
| "跌了很多了" | "跌得多不等于要涨。什么信号出现说明见底了？" |

### 认知评估标准

| 分数 | 标准 |
|------|------|
| 8-10分 | 了解行业周期、竞争格局、催化剂逻辑，有时间框架 |
| 5-7分 | 能说出催化剂，但缺少具体时间框架或触发条件 |
| 1-4分 | 只说"政策支持""概念热""跌多了"，没有实质逻辑 |

## 隐性追问：退出纪律、仓位、认知偏差

### 为什么需要隐性追问

三问法覆盖的是"买"的逻辑，但投资还需要"卖"的纪律和"风险管理"的意识。

### 追问内容

#### 1. 退出纪律

- "如果买入后跌了20%，你会怎么办？"
- "什么情况下你会认错出局？"
- "你的止损线设在哪？"

#### 2. 仓位管理

- "你打算用多大仓位买这只股票？"
- "是一次性买入还是分批建仓？"
- "这只股票在你整个组合中占多少比例？"

#### 3. 认知偏差检测

- "你有没有只看了支持你观点的信息？"
- "你是不是因为最近涨了才想买？（近因效应）"
- "你是不是因为已经研究了这么久，所以觉得必须买？（沉没成本）"

### 认知评估标准（风险认知）

| 分数 | 标准 |
|------|------|
| 8-10分 | 有止损纪律、仓位规划、能识别自己的确认偏误 |
| 5-7分 | 有止损意识，但缺少具体执行标准 |
| 1-4分 | 从没想过止损、全仓梭哈、意识不到自己的偏差 |

## 自适应追问力度

根据用户回答质量动态调整追问深度：

### 高质量回答（8-10分水平）

**模式**：温和探讨
**追问轮数**：1-2轮
**语气**："说得不错，我再补充一个角度…"

### 中等质量回答（5-7分水平）

**模式**：投资教练
**追问轮数**：2-3轮
**语气**："你的判断是X，支撑这个判断的关键假设是什么？"

### 低质量回答（1-4分水平）

**模式**：严格追问
**追问轮数**：3-4轮
**语气**："你说的这些，换个公司也成立。这家公司独特在哪？"

## 常见误区与纠正

| 误区 | 纠正 |
|------|------|
| "行业好=公司好" | 行业好只是必要条件，不是充分条件。要找出公司独特的竞争优势。 |
| "PE低=便宜" | 低PE可能是因为增长停滞、风险上升。要结合增速和质量看。 |
| "跌多了=要涨" | 跌得多不等于见底。要找出见底的信号和催化剂。 |
| "机构买=要涨" | 机构买入可能是滞后指标。要理解机构买入的逻辑是否可持续。 |
| "研究久=必须买" | 研究是为了搞懂，不是为了买。不懂就不要买。 |
